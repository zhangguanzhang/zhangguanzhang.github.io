иҝӣе…Ҙзӣ®еҪ•е№¶иҝҗиЎҢdemo

жҹҘзңӢcmd/root.goеҸ‘зҺ°е‘Ҫд»Өзҡ„й•ҝзҹӯеё®еҠ©ж–Үеӯ—пјҢеӯ—йқўзңӢзҡ„иҜқиҜҙдҪҝз”ЁappиҝҗиЎҢпјҢз„¶еҗҺз»ҷappе‘Ҫд»Өж·»еҠ й•ҝзҹӯзҡ„её®еҠ©иҜҙжҳҺж–Үеӯ—

иҝҗиЎҢжҹҘзңӢ

ж·»еҠ еӯҗе‘Ҫд»ӨиҫҫеҲ°жғіиҰҒзҡ„еұӮзә§е…ізі»

жҢүз…§з”ҹжҲҗеҷЁз”ҹжҲҗзҡ„д»Јз ҒдјҡжҳҜдёӢйқўзҡ„з»“жһ„пјҢжүҖд»Ҙз”ҹжҲҗеҗҺжҲ‘д»¬йңҖиҰҒдҝ®ж”№remove.goйҮҢзҡ„д»Јз Ғ

дёҠйқўе№¶жІЎжңүиҫҫеҲ°жҲ‘д»¬йў„жңҹзҡ„иҫ“еҮәпјҢжҲ‘д»¬жңҹжңӣжҳҜgo run main.go appзҡ„ж—¶еҖҷиҫ“еҮәжңҖеҗҺзҡ„--helpиҝҷж ·зҡ„е‘Ҫд»Өеё®еҠ©жҸҗйҶ’з”ЁжҲ·гҖӮиҝҷж ·жңүдёӨз§Қе®һзҺ°ж–№жі•пјҢдёҖз§ҚжҳҜжҠҠvar appCmd = &cobra.Commandзҡ„ж—¶еҖҷRunеҲ жҺүпјҢжҲ–иҖ…еғҸдёӢйқўж”№жҲҗRunEпјҡ
